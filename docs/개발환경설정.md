# 개발 환경 설정

## 사전 요구사항

- JDK 21
- Gradle 9.x

## 환경 구분

| 프로파일 | DB | 용도 |
|----------|-----|------|
| **local** (기본) | H2 (메모리) | 로컬 개발 |
| **prod** | MySQL (AWS RDS) | 운영 |

## 로컬 환경 (local)

- **DB**: H2 인메모리 (별도 설치 불필요)
- **실행**: 프로파일 미지정 시 기본 적용

```bash
./gradlew bootRun
```

또는 명시적으로:

```bash
./gradlew bootRun --args='--spring.profiles.active=local'
```

### OAuth2 (카카오/네이버/구글 로그인 테스트)

로컬에서 OAuth 로그인을 테스트하려면 **환경변수**를 설정해야 합니다. (기본값 `local-placeholder`는 카카오 API에서 거부됨)

```bash
# 터미널에서 실행 시
export KAKAO_CLIENT_ID="카카오_개발자_콘솔에서_발급한_REST_API_키"
export KAKAO_CLIENT_SECRET="카카오_개발자_콘솔에서_발급한_Client_Secret"
./gradlew bootRun
```

또는 한 줄로:

```bash
KAKAO_CLIENT_ID=your_client_id KAKAO_CLIENT_SECRET=your_client_secret ./gradlew bootRun
```

**카카오 개발자 콘솔**: https://developers.kakao.com → 내 애플리케이션 → 앱 키 → REST API 키, Client Secret

### H2 콘솔

- URL: http://localhost:8080/h2-console
- JDBC URL: `jdbc:h2:mem:minton_match`
- User: `sa`, Password: (비워두기)

## 운영 환경 (prod)

- **DB**: AWS RDS (MySQL)
- **실행**:

```bash
./gradlew bootRun --args='--spring.profiles.active=prod'
```

### 환경변수 (운영 배포 시)

| 변수 | 설명 |
|------|------|
| DB_URL | (선택) 전체 JDBC URL. 미설정 시 기본 RDS URL 사용 |
| DB_USERNAME | DB 사용자명 |
| DB_PASSWORD | DB 비밀번호 |
