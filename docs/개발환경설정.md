# 개발 환경 설정

## 사전 요구사항

- JDK 21
- MySQL 8.x (또는 호환 DB)
- Gradle 9.x

## 데이터베이스 설정

1. MySQL에서 데이터베이스 생성:

```sql
CREATE DATABASE minton_match CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

2. `src/main/resources/application.yml`에서 DB 접속 정보 수정:

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/minton_match?...
    username: your_username
    password: your_password
```

3. 개발 시 `application-dev.yml` 프로파일 사용 (스키마 자동 생성):

```bash
./gradlew bootRun --args='--spring.profiles.active=dev'
```

## 실행

```bash
./gradlew bootRun
```
